# docker-rasenmaeher-integration 2.6.0

**Release date:** 2026-02-22

## TL;DR
### Video streaming using MediaMTX
- Auto-import for **OpenTAK ICU**: Users can now auto-import streaming settings into OpenTAK ICU, an open source TAK-integrating streaming app, with a single button click.  This instead of manually copying each field (protocol, address, port, path, username, password). Requires OpenTAK ICU v1.5.4+. **Starting streaming is now very easy, 'Press this button and go.'**
- **UAS Tool import**: Added a one-click `tak://` deep link to import all streaming configuration into UAS Tool automatically. **Starting streaming from an UASTool integrating drone is now very easy, 'Press this button and go.'**
- **Stream package downloads**: New `useStreamPackages` hook and UI for downloading pre-configured stream packages (ATAK RTMPS, Browser HLS, VLC SRT, VLC HLS) as ready-to-use files from the Live page.
- **SRT URL fix**: Removed leading slash from SRT `streamid` parameter (`publish:live/icu/...` instead of `publish:/live/icu/...`).

  - https://github.com/pvarki/python-mediamtx-rmmtxauthz/pull/38
  - https://github.com/pvarki/python-rasenmaeher-api/pull/143
- **Updated onboarding**: Added a new onboarding step explaining auto-import, and renamed manual setup steps for clarity ("Starting a Stream Manually - Step 1/2/3").
- **Localization**: All new UI strings translated into English, Finnish, and Swedish.

### Battlelog (integrated product) improvements
- Timeline feature, display events in timeline.
- UI fixes
- Battlelogs product card can now be toggled on/off (default off) from the env variables. This will only affect product card in the main app list, card is visible in the admin app anyway.

### Audit logging feature
- Logging covers CRUD operations by rmapi
- Uses libpvarki 2.2+ for logging
- Context required by audit logging provided by utility module at utils. In the future this will be done by libpvarki.
- Logging spec is written and later posted to Developer documentation.

No breaking changes.

---

## Release Goals
### Streaming
- Dramatically simplifies stream setup**: Previously users had to manually copy 6+ fields (protocol, address, port, path, username, password) into OpenTAK ICU settings. Now it's a single button tap via `opentakicu://import` deep link.
- **Reduces user errors**: Auto-import eliminates typos and misconfiguration during manual stream setup, which was a common source of support requests.
- **Enables UAS Tool integration**: One-click import configures 16 UAS Tool preferences at once (SRT destination, callsign, broadcast settings, etc.), unlocking UAS streaming workflows that were previously tedious to set up.
- **Stream packages for advanced users**: Downloadable ATAK RTMPS, Browser HLS, VLC SRT, and VLC HLS packages allow quick stream consumption across different platforms and players without manual URL construction.

### Battlelog
- Improve the product toward more usage

### Audit logging
- Improve Deploy App security posture & accountability
