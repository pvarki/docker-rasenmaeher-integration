# docker-rasenmaeher-integration 2.3.3

**Release date:** 2026-02-22

## TL;DR

- Added new first time guides to takintegration.
- Added unsupported browser warning.
- Fixed MediaMTX showing the guide on every stream.
- Fixed all product integrations sometimes refreshing the UI .

No breaking changes.

---

## Release Goals

- Drawer based guides help users use tak and especially the auto import on android.
- Warning for some obsecure browsers should result in better user experience.
- Integration UI no longer reloading / losing state reduce user confusion.
